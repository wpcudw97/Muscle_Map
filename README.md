# Muscle_Map

## サイト概要
### サイトテーマ
なりたい自分になるための情報を検索できたり、おすすめの筋トレメニュー、器具を投稿できたりするコミュニティサイト


### テーマを選んだ理由
なりたい体型を目指すには、何を何回どのペースでやればいいのか、何を食べればいいのか知ることができるサービスがあれば便利だと思い、このテーマにしました。


### ターゲットユーザ
自分を変えたい人。

### 主な利用シーン
筋トレメニューを検索するとき。おすすめの筋トレ器具について投稿、共有するとき。
